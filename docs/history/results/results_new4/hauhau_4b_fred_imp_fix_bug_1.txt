GOAL: Resolve intermittent authentication failures in the user login flow where valid credentials result in a 401 Unauthorized error after successful initial attempts, specifically occurring when session tokens are regenerated during API calls.

FILES: `src/auth/login.service.ts`, `src/middleware/session.interceptor.ts`, `tests/e2e/login.spec.ts`

STEPS:
1. Analyze the token validation logic in `login.service.ts` to identify if it is checking for an expired timestamp or a revoked status flag incorrectly during subsequent requests.
2. Update `session.interceptor.ts` to ensure the JWT payload includes a 'refresh_at' claim that triggers regeneration only when the current session age exceeds 15 minutes, preventing premature token invalidation.
3. Modify the login endpoint in `login.service.ts` to handle "expired but not revoked" tokens by automatically issuing a new access token and updating the user's last_login timestamp without requiring re-authentication.
4. Implement retry logic with exponential backoff (max 3 attempts) within the client-side auth module if a 502 Bad Gateway or transient 401 is received immediately after login.
5. Update `login.spec.ts` to include test cases for: valid credentials + long idle time, invalid password vs expired token scenarios, and concurrent session handling.

EDGE CASES:
- User has multiple active sessions; ensure revocation of one does not invalidate others unless explicitly requested by the user via "logout all".
- Network latency causes race conditions between token refresh API call and client-side timeout logic (implement optimistic locking).
- Database lockouts during high concurrency login attempts causing temporary unavailability.

ACCEPTANCE:
- All existing unit tests pass with zero failures.
- E2E test suite passes for the "login after 30 minutes of inactivity" scenario without manual intervention.
- Authentication latency remains under 500ms for valid credentials on average.
- No increase in error rate (4xx/5xx) observed during load testing with 1,000 concurrent users.
