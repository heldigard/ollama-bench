# Handoff Note: Flaky Test Investigation and Infrastructure Updates

## Decisions

### Logging Framework Migration
- **Decision**: Switch from Python's standard `logging` module to [`loguru`](https://github.com/Delgan/loguru) for structured logging
- **Rationale**: Loguru provides better performance, automatic formatting of timestamps/levels/messages without boilerplate code, and improved stack trace handling

### Build Backend Selection  
- **Decision**: Use hatchling as the build backend instead of setuptools
- **Rationale**: Hatchling offers significantly faster installation times for large projects due to optimized dependency resolution algorithms (PEP 621 support)
